<h2>💻 Sobre o projeto</h2>

📰 Bikcraft - Site desenvolvido no curso de Web Design Completo da Origamid e Wordpress como CMS. 

<p>No curso de Web Design Completo foi construído o site do zero, desde o wireframe, passando pelo design, até o código em si. Foi feita uma abordagem básica de HTML, CSS e Javascript. O posicionamento dos elementos foi realizado principalmente através de um grid pré-criado de 12 colunas, apenas para facilitar. Porém, toda a forma para a construção desse grid é explicada. Outras técnicas como float e position são utilizadas. Nesse momento não foram utilizados flexbox nem o grid-layout. O Javascript foi utilizado apenas para acrescentar o scroll de uma seção da página e algumas animações, não sendo muito explorado, pois não era o foco do curso. As funções foram explicadas brevemente e disponilizadas para uso via download.</p>
<p>Já no curso de Wordpress como CMS, aprendemos como utilizar as ferramentas do Wordpress para gerenciar o conteúdo sem a necessidade de editar o código diretamente, servindo também para o cliente poder fazer alterações independente do desenvolvedor. Para isso, todo o html foi convertido para PHP, de forma que os campos do site se tornassem dinâmicos.</p>
<p>Por fim, é ensinado como colocar o site no ar em um servidor e o básico de um formulário funcional também via PHP.</p>
